# CVAT 라벨링 실습용 이미지

CVAT(Computer Vision Annotation Tool)에서 **직접 라벨링을 체험**하기 위한 혈액 도말 이미지입니다.

- **출처**: [BCCD_Dataset](https://github.com/Shenggan/BCCD_Dataset) — 라이선스 **MIT**.
- **용도**: CVAT에 업로드해 박스/마스크를 직접 그려보는 실습 (session2 3단계의 "CVAT + SAM 통합" 참고).
- **구성**: 혈액 도말 JPEG 6장 (`BloodImage_00000.jpg` ~ `BloodImage_00005.jpg`).

> **참고(예외)**: 공통 규칙([CONVENTIONS.md](../../CONVENTIONS.md) 규칙 2)은 보통 이미지를 레포에 넣지 않고 코드로 내려받습니다.
> 이 폴더는 **CVAT 실습 편의를 위해 소량을 의도적으로 포함**한 예외입니다.
